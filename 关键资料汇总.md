## 如何更新网站

### 方法1：直接在 GitHub 网页操作
1. 打开 https://github.com/okeming-tech/oukeming-smart-desk
2. 点击要修改的文件 → 点铅笔图标编辑 → 或上传新文件替换旧文件
3. 编辑完成后点击 "Commit changes"
4. GitHub Pages 会自动重新部署，1-2分钟生效

### 方法2：删掉重新生成
告诉我需要改什么内容，我帮你重新生成网站并上传。

### 方法3：添加视频
- 如果视频小于100MB，可以直接上传到仓库，在 index.html 中添加 `<video>` 标签
- 推荐先把视频传B站/YouTube，再嵌入网站，加载更快
